Block level Element

    in this type of tags all contents are displayed in line by line
    And it will take pool width of the parent" body"
    Here we can prove height and width
    Example:h,p and div tag

Inline level Element

    in this type of tags all contents are displayed in Single line
    And it will take pool width is whatever the content will take space
    Here we can't prove height and width
    Example:b,u and i tag
